# Neo-tree 插件快捷键

| 快捷键/命令 | 作用 | 说明 |
| - | - | - |
| `<C-n>` | 打开文件系统树并定位当前文件 | Neo-tree 全局映射 |
| `<leader>e` | 打开文件系统树（左侧） | Neo-tree 全局映射 |
| `<leader>bf` | 浮动窗口显示缓冲区树 | Neo-tree 全局映射 |
| `<leader>gs` | 打开 Git 状态视图（左侧） | Neo-tree 全局映射 |
| `<Esc>`（Neo-tree 窗口中） | 关闭 Neo-tree 窗口 | Neo-tree Buffer 映射 |
| `<CR>` / `<2-LeftMouse>` | 打开当前节点 | Neo-tree 默认映射 |
| `<space>` | 展开/折叠目录 | Neo-tree 默认映射 |
| `P` | 预览当前节点（浮动窗口） | Neo-tree 默认映射 |
| `S` / `s` / `t` | 水平分屏 / 垂直分屏 / 新标签页打开文件 | Neo-tree 默认映射 |
| `a` / `A` | 新建文件 / 新建目录 | Neo-tree 默认映射 |
| `d` / `r` | 删除 / 重命名 | Neo-tree 默认映射 |
| `y` / `x` / `p` | 复制 / 剪切 / 粘贴 | Neo-tree 默认映射 |
| `c` / `m` | 复制到指定路径 / 移动到指定路径 | Neo-tree 默认映射 |
| `H` / `.` / `<BS>` | 切换隐藏文件 / 设置根目录 / 返回上级目录 | Neo-tree 默认映射 |
| `/` / `D` | 模糊搜索 / 目录内模糊搜索 | Neo-tree 默认映射 |
